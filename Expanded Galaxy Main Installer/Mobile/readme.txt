To install for Android

Select the Android version from the dropdown

Paste the location of this directory in the directory dropdown

Install to this directory, then copy everything over to /Home/Android/data/com.aspyr.swkotor2/files/



To install for iOS

Install to this directory, then copy everything over using iTunes.

On iOS, they are placed in iTunes > Apps > Documents, in the same spot that the ‘saves’ folder goes.
